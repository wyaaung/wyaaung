## 🚀 Engineering Philosophy & Current Focus

I’m a **Software Engineer** specializing in high-throughput, low-latency distributed systems and developer-focused platform engineering, with a focus on **Zero Trust security**, **edge-native compute**, and **event-driven microservices**. I design and build resilient systems that can withstand regional outages, absorb massive traffic spikes, and deliver sub-millisecond response times globally. 

My day-to-day involves moving heavy origin compute to the edge using **Cloudflare Workers**, **Fastly Compute@Edge**, and **WebAssembly (Wasm)** to reduce TTFB and distribute execution closer to users, while helping build active-active, multi-region platforms powered by advanced **CDN routing** and replicated state. Beyond product engineering, I build the platforms behind the platforms, from enterprise control planes to core Java execution frameworks.

## 📈 Engineering Metrics I Optimize For
* 📉 **P99 Latency:** Relentlessly optimizing the tail latency at the edge through programmatic VCL routing, CDN shield topology design, and healthcheck-aware failover — not reactive alerting.
* ⏱️ **Time to First Byte (TTFB):** Pushing compute and cached data as close to the user as physically possible via distributed edge networks.
* 🚀 **Developer Velocity:** Reducing microservice bootstrap time from days to minutes through world-class IDP frameworks.
* 🔄 **Mean Time to Recovery (MTTR):** Architecting self-healing, automated systems over relying on manual intervention.
