## 🚀 Engineering Philosophy & Current Focus

I’m a **Software Engineer** specializing in high-throughput, low-latency distributed systems and developer-focused platform engineering, with a focus on **Zero Trust security**, **edge-native compute**, and **event-driven microservices**. I design and build resilient systems that can withstand regional outages, absorb massive traffic spikes, and deliver sub-millisecond response times globally.

Beyond product engineering, I build the platforms behind the platforms, from enterprise control planes to core Java execution frameworks.
